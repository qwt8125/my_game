# v0.5.8 地图缩放与拖拽视角计划

## 1. 优化目标

- 让玩家在点位聚合之后还能进一步放大查看局部地图，降低小屏幕误触和读点成本。
- 支持拖拽地图视角，让玩家可以在放大后移动观察区域。
- 保持地图坐标事实仍由后端场景接口提供，本轮缩放和拖拽只影响前端视图。

## 2. 范围

- 地图工具栏新增：
  - 放大
  - 缩小
  - 重置视角
- 地图画布支持鼠标/触控拖拽视角：
  - 只在拖拽空白画布时平移视图。
  - 点击点位、聚合菜单和工具栏不触发拖拽。
  - 拖拽后不触发空白移动角色。
- 角色空白点击移动、点位聚合、移动端日志抽屉继续保留。
- 静态资源版本号更新，避免浏览器缓存旧脚本和样式。

## 3. 不做范围

- 不新增后端坐标保存接口。
- 不保存缩放/视角到数据库。
- 不做滚轮缩放、双指缩放和惯性拖拽。
- 不改变地图事件、NPC、战斗和任务推进规则。

## 4. 验收标准

- 点击放大/缩小后，地图内容按比例缩放。
- 点击重置视角后，地图回到 100% 缩放和居中视角。
- 放大后拖拽地图空白区域可平移视角。
- 短点击空白区域仍可移动前端角色标记；拖拽不会误触发移动。
- 点位、聚合菜单和日志抽屉仍可正常操作。
- `node --check game/src/main/resources/static/app.js`、`mvn test`、`mvn package -DskipTests` 通过。
