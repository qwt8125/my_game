# v0.5.2 每日地图探索内容扩充计划

## 1. 背景

v0.5.1 已支持地图事件的经验、金币、物品奖励和 `resetType=daily` 每日重置规则，但当前只有新手村外配置了每日探索样板点。为了让玩家在不同等级段进入地图后都有稳定的日常收益，需要把每日探索点扩展到更多地图。

## 2. 目标

- 为 8 张地图补齐每日探索收益点，每张地图至少有 1 个 `resetType=daily` 奖励事件。
- 每日点奖励随地图等级段递增，保持轻量收益，不替代打怪、任务和 BOSS。
- 每日点使用已有地图事件状态能力，当天触发后冷却到次日 00:00。
- 补充少量材料配置，让不同地图的每日点有差异化掉落。

## 3. 配置策略

- 低等级地图给少量经验、金币和基础材料。
- 中等级地图给略高经验金币，并混入矿石、毒囊、骨片等材料。
- 高等级地图给更高每日收益，但仍低于主线任务和 BOSS 奖励。
- 每日探索点都保留 `requiredTaskIds`，避免玩家越级拿高等级地图日常奖励。

## 4. 验收标准

- `map-events.json` 中 8 张地图都至少有 1 个 `resetType=daily` 的 `reward` 事件。
- 新增奖励物品都存在于 `items.json`，配置加载校验不报错。
- `GET /api/maps/{mapId}/scene` 能展示新增每日点。
- 每日点触发后返回经验、金币和物品奖励，并进入冷却。
- `mvn test` 和 `mvn package -DskipTests` 通过。
