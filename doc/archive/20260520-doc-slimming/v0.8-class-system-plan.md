# v0.8 职业系统与职业分化计划

## 1. 目标

把当前单职业 `warrior` 扩展为战士、法师、道士三职业，并让职业影响初始属性、成长、可学习技能和前端角色创建选择。

## 2. 职业定位

### 战士 `warrior`

- 高生命、高防御、近战爆发。
- 推荐技能：烈斩、破甲刺击、护体战意、逐风步。

### 法师 `mage`

- 高攻击、低防御、技能爆发。
- 推荐技能：火符术、雷光术、星火燎原、灵能贯通。

### 道士 `taoist`

- 均衡成长、持续伤害、治疗和召唤预留。
- 推荐技能：施毒术、回春术、玄灵符、灵兽契印。

## 3. 配置字段

`classes.json`：

```json
{
  "id": "warrior",
  "name": "战士",
  "description": "生命和防御优秀，擅长近身爆发。",
  "hp": 140,
  "attack": 13,
  "defense": 6,
  "attackSpeed": 96,
  "hpPerLevel": 24,
  "attackPerLevel": 3,
  "defensePerLevel": 2,
  "attackSpeedPerLevel": 1
}
```

## 4. 后端范围

- 新增 `classes.json` 配置和加载校验。
- 创建角色支持 `warrior`、`mage`、`taoist`。
- 角色基础属性和升级成长来自职业配置。
- 旧角色默认继续视为 `warrior`。
- 技能按 `className` 过滤。

## 5. 前端范围

- 创建角色时提供职业选择。
- 角色页展示职业中文名和定位。
- 技能页展示职业技能。

## 6. 不做范围

- 不做转职。
- 不做职业专属装备限制。
- 不做职业剧情分线。

## 7. 验收标准

- 新角色可选择三职业。
- 不同职业初始属性和升级成长不同。
- 不同职业看到的技能池不同。
- 旧角色不受影响。
